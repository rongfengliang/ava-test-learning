# ava js test framework learning

> use typescripts 

## how to running

* install deps

```code
yarn
```

* build libs for typescript

```code
yarn build:live
```

* live test

```code
yarn test:live
```

## snapshot test

*  test

```code
yarn test
```

* rest cache

```code
npx ava  --reset-cache
```

* update snapshot 

```code
npx ava -u
```
